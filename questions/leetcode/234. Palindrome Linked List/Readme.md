## [234. Palindrome Linked List](https://leetcode.com/problems/palindrome-linked-list/)
 Given a singly linked list, determine if it is a palindrome.

 ## Follow-up
 Could you do it in O(n) time and O(1) space?
 
 ## Sample input
 ### Example 1
 ```c
 Input: 1->2
 Output: false
 ```

 ### Example 2
 ```c
 Input: 1->2->2->1
 Output: true
 ```
