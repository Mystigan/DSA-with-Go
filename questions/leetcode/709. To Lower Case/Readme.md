## [709. To Lower Case](https://leetcode.com/problems/to-lower-case/)
 Implement function ToLowerCase() that has a string parameter str, and returns the same string in lowercase.

 ## Sample input
 ### Example 1
 ```c
 Input: "Hello"
 Output: "hello"
 ```
 ### Example 2
 ```c
 Input: "LOVELY"
 Output: "lovely"
 ```
