## [83. Remove Duplicates from Sorted List](https://leetcode.com/problems/remove-duplicates-from-sorted-list/)
 Given a sorted linked list, delete all duplicates such that each element appear only once.

 ## Sample input
 ### Example 1
 ```c
 Input: 1->1->2
 Output: 1->2
 ```
 ### Example 2
 ```c
 Input: 1->1->2->3->3
 Output: 1->2->3
 ```
